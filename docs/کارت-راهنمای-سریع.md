# کارت راهنمای سریع هنرستان هادی

## ورود سریع
- **آدرس پنل:** `http://localhost:3000/admin`
- **نام کاربری:** `admin`
- **رمز عبور:** `admin123`

## میانبرهای سریع

### اضافه کردن محتوا:
| محتوا | مسیر |
|--------|------|
| خبر جدید | پنل → اخبار → خبر جدید |
| تصویر جدید | پنل → گالری → تصویر جدید |
| دوره جدید | پنل → دوره‌ها → دوره جدید |
| استاد جدید | پنل → اساتید → استاد جدید |
| اثر جدید | پنل → آثار هنرجویان → اثر جدید |
| رویداد جدید | پنل → رویدادها → رویداد جدید |
| نظر جدید | پنل → نظرات → نظر جدید |

### فرمت‌های تصویر:
- JPG, PNG, GIF, WebP
- حداکثر حجم: 10 مگابایت

### نکات سریع:
- ✅ برای نمایش محتوا در سایت، تیک «نمایش» را بزنید
- ✅ آثار «برتر» در صفحه اصلی نمایش داده می‌شوند
- ✅ از تصاویر با کیفیت بالا استفاده کنید
- ✅ تغییرات بلافاصله اعمال می‌شوند

### عیب‌یابی سریع:
- صفحه بارگذاری نمی‌شود → رفرش کنید
- آپلود کار نمی‌کند → حجم فایل را کاهش دهید
- تغییرات ذخیره نمی‌شوند → اتصال اینترنت را بررسی کنید
